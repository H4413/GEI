\subsection{Les règles de suivi}

On établit les règles de suivi suivantes :

\begin{itemize}
\item Les tâches sont distribuées par mail par le Chef de Projet
\item Un rappel de l'ensemble des tâches assignées aux différents membres
de l'équipe est disponible pour tous sur le tableau de bord de l'équipe
(google Spreadsheet)
\item Chacun est tenu de mettre à jour les tâches le concernant.
\item Le CdP valide les drafts associés aux tâches sur le fond
\item Le RQ valide les livrables intermédiaires associés aux tâches sur la
forme
\end{itemize}

Par ailleurs l'ensemble du travail de l'équipe est versionné en utilisant
git associé au service github.com, ce qui permet de le mettre à disposition de tous
via Internet, de se prémunir efficacement contre les effacements
malencontreux de dossiers et de versionner facilement drafts, livrables
intermédiaires et livrables.
